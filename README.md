\# Netflix Movies \& TV Shows EDA Project



\## Project Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on

the Netflix Movies and TV Shows dataset and identify meaningful insights using Python.



\## Tools Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Google Colab



\## Project Steps

1\. Data Loading and Inspection

2\. Data Cleaning

3\. Exploratory Data Analysis

4\. Data Visualization

5\. Insights Generation



\## Visualizations

\- Bar Chart

\- Pie Chart

\- Line Chart

\- Histogram

\- Scatter Plot

\- Heatmap



\## Key Findings

\- United States contributes the most content.

\- Movies are more common than TV Shows.

\- Netflix content additions increased rapidly after 2015.

\- Most movies are 80–120 minutes long.

\- Netflix focuses on newer content while maintaining older titles.

\- Missing values were mainly found in director, cast, and country columns.
